## 📦 Installation & Local Development

```bash
# 1. Clone this repo
git clone https://github.com/son-u/faiz-dev-co.git
```
```bash
# 2. Navigate into project directory
cd faiz-dev-co
```
```bash
# 3. Install dependencies
npm install
```
```bash
# 4. Start the Vite dev server
npm run dev
```
